# Car Rental — 汽车租赁平台（Java 后端服务）

一个面向微信小程序的汽车租赁平台后端系统，采用 **Spring Boot 3 + MyBatis-Plus + MySQL** 架构，支持用户自助租车、微信支付、后台车辆管理及订单处理，覆盖租车全链路业务场景。

### 双端设计

系统分为「**用户端**」和「**管理端**」两个独立模块，共享底层服务，通过不同的 JWT 拦截器实现鉴权隔离：

- **用户端** (`com.sunl19ht.client.*`)：小程序用户登录、浏览车辆、下单、支付、查看订单
- **管理端** (`com.sunl19ht.*`)：车辆管理、订单处理、还车验车、退款审核

## 业务功能

### 用户端（租车客户）

- **微信一键登录** — `wx.login()` 换取 OpenID，无密码登录，JWT 签发 Token
- **车辆浏览** — 按取还车时间 + 车型筛选空闲车辆，展示含周末/平日差异化价格
- **下单租车** — 选择取还车方式（自取/送车上门），填写地址、身份证信息，选择保险方案
- **智能计价** — 基础租金 + 服务费 + 保险费 + 送车费（调用外部距离服务计算），透明展示价格明细
- **微信支付** — JSAPI 下单 → 唤起小程序支付 → 异步回调更新订单状态
- **订单管理** — 按状态分类查看订单，支持取消（取车前 4 小时免费）
- **首页轮播** — 运营位 Banner 展示

### 管理端（运营人员）

- **员工登录** — 同微信小程序登录，员工表独立存储
- **车辆管理** — 车辆 CRUD、上下架、设置周末/平日价格及划线价、多图上传
- **临时停运** — 按时间段下线车辆，定时任务到期自动恢复
- **工作台** — 待取车 / 待还车 / 已完成订单数量实时统计
- **订单处理** — 取车时上传身份证、驾驶证、车辆照片/视频 → 自动流转状态；还车时上传车况 → 验收完成
- **退款处理** — 微信支付退款接口，原路退回

### 定时任务

| 任务 | 频率 | 说明 |
|------|------|------|
| 订单超时取消 | 每 5 秒 | 未支付超过 15 分钟的订单自动取消 |
| 车辆自动恢复 | 每 5 秒 | 临时停运到期的车辆自动上架 |

### 订单生命周期

```
待支付 → 已支付/待取车 → 待还车 → 已完成
  │                        │
  └── 取消（退款） ←────────┘
```

## 快速启动

### 环境要求
- JDK 17+
- Maven 3.8+
- MySQL 8.0+
- 微信商户平台账号（支付功能需要）

### 构建 & 运行

```bash
# 1. 修改 application.yaml 中的数据库连接、微信支付密钥、COS 密钥等配置

# 2. 构建
mvn clean package -DskipTests

# 3. 启动
nohup java -jar target/car-rental-1.0-SNAPSHOT.jar > app.log 2>&1 &
```

服务默认运行在 **9090** 端口，API 文档访问 `http://localhost:9090/swagger-ui.html`。

## 技术亮点

1. **微信支付 API v3 完整对接** — JSAPI 下单、RSA 二次签名、AES 回调解密、退款接口全部覆盖
2. **双端 JWT 鉴权** — 用户端与管理端使用独立的 JWT Secret 和拦截器链，职责清晰
3. **复杂计价模型** — 基础租金 × 用车时长 + 服务费 + 保险费分级 + 送车地理位置距离计算，支持划线价
4. **订单状态机** — 支付回调、取/还车图片上传等事件驱动状态自动流转
5. **定时任务兜底** — 超时未支付订单自动取消、停运车辆到期自动恢复，保障数据一致性
6. **文件存储双适配** — 同时集成腾讯云 COS 和阿里云 OSS，可灵活切换
